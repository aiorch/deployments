# Deployments
Deployment manifests and virtual environment for components to perform LLM inference

## Getting Started
 ```
 Start the quadrant database
 ${CWD}/quadrant/quadrant.sh
 
 ```

 ```
 Start the inference server
 ${CWD}/zephyr-7B/model.sh

 ```

 ```

 Run the query
 ${CWD}/llm_chain.py

 ```
